## Trạng thái hệ thống

Hệ thống hiện có:

- Frontend Next.js cho admin và seller.
- Backend NestJS API.
- MongoDB Atlas.
- JWT authentication.
- Role-based authorization.
- Socket.IO realtime notifications.
- Swagger API docs.
- Health endpoint.
- Demo seed data.
- Backend unit and e2e tests.
- Dockerfile và Docker Compose theo hướng dùng MongoDB Atlas.
